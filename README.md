# ☕ Guia de Estudos: Ecossistema Java

Este projeto é um portal didático focado na linguagem Java, desenvolvido para consolidar conhecimentos em tecnologias Web (HTML5 e CSS3) e servir como fonte de consulta sobre o universo Java, seus frameworks e ferramentas.

O projeto foi inspirado nas aulas de HTML5 e CSS3 do [Curso em Vídeo](https://www.cursoemvideo.com/), ministrado pelo professor **Gustavo Guanabara**.

🔗 **[Acesse o projeto online aqui](https://jp-rocha23.github.io/Site-Java/)**


## 🚀 Sobre o Projeto

O site foi estruturado para guiar desde o iniciante até o desenvolvedor que busca entender as ferramentas modernas do mercado. Ele está dividido em seções estratégicas:

* **Página Inicial:** Introdução histórica e o conceito "Write Once, Run Anywhere".
* **Instalação:** Guia prático de configuração do JDK e comparação detalhada entre as principais IDEs (Eclipse vs. IntelliJ).
* **Notícias:** Atualidades sobre Java 21, Spring Boot 3 e a integração de IA com Java.
* **Frameworks:** Explicação de conceitos avançados como *Inversão de Controle (IoC)* e visão geral de Spring Boot, Hibernate, Quarkus e JUnit.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Design responsivo, estilização de tabelas técnicas e componentes de interface.
* **GitHub Pages:** Hospedagem e deploy contínuo do projeto.

## 💡 O que foi implementado?

* **Layout Flexbox:** Header com logo centralizado e menu de navegação em linha.
* **Estilização de Código:** Bloco de código estilizado para simular uma IDE (VS Code) com foco na legibilidade.
* **Links "Easter Egg":** Uso de classes CSS para criar links discretos que herdam o estilo do texto pai.
* **Responsividade:** Container de vídeo que se ajusta automaticamente a diferentes tamanhos de tela.

## 🎓 Contexto Acadêmico e Profissional

Este repositório faz parte do meu portfólio como estudante de **Ciência da Computação na UERJ**. O objetivo é aplicar boas práticas de desenvolvimento front-end enquanto documento o aprendizado em tecnologias de backend que utilizo no dia a dia, como C#, Oracle Database e, claro, Java.
